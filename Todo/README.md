# Extension package (`Todo/`)

This directory is the unpacked extension payload (popup, scripts, styles, icons, and [`manifest.json`](manifest.json)).

For full documentation, installation options, and features, see the [repository README](../README.md).

**Load this folder only:** In `chrome://extensions`, choose **Load unpacked** and select this `Todo` directory. Chrome will use the manifest in this folder.

**Load the repo root:** See the root README—use the parent folder so Chrome picks up the root `manifest.json` that references paths under `Todo/`.
